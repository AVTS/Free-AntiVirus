How to make mini - antivirus

Hello there! My name is Yura K. and I tell you how to make mini antivirus in Visual Basic

We need:

2 buttons; 1 richtextbox; 1 openfiledialog

Step 1

First we need to does 1 left click on button 1 and set its name to Choose Folder. Then, we need to does 1 left click on button 2 and set its name to Delete File.

Step 2

First we need to does 1 left click on button 2 and set value enabled to false in object properties. Then, we need to does 1 left click on richtextbox1 and set value 'visible' to false in object properties.

Step 3 - coding!

First we need to does 2 left clicks on button 1 and write this code: OpenFileDialog1.ShowDialog() If OpenFileDialog1.SelectedFile = "" Then MsgBox("You does not select file to scan! Please, try again.", MsgBoxStyle.Information) Else Dim A1 = new StreamReader(OpenFileDialog1.SelectedFile) RichTextBox1.Text = A1.ReadToEnd() If RichTextBox1.Contains("virus") Then MsgBox("A Virus was found!") Button2.Enabled = "True" End If If RichTextBox1.Contains("trojan") Then MsgBox("A Virus was found!") Button2.Enabled = "True" End If End If Warning! You need add viruses after RichTextBox1.Text = A1.ReadToEnd() (examples is after it.) Then, we need to does 2 left click on button2 and write this code: My.Computer.FileSystem.DeleteFile(OpenFileDialog1.SelectedFile) MsgBox("Virus was deleted. Your computer is now safe. This mini antivirus was built with http://uaavts.p.ht/") Have fun!
